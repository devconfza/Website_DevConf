About the talk

In this talk, we'll dispel the common myth that you need a room full of PhDs to implement data science and machine learning into your software product. We'll present data science and machine learning as another tool that everyone involved with building software should have in their toolbox. We'll talk about the concepts, tools, and resources that a software developer needs to get familiar with, the value that data science and machine learning can bring to a software product, and how to get started doing it yourself.

About the speaker

Helge is a data scientist with 6+ years of experience in the practical implementation of products and services across a multitude of industries including insurance, banking, retail, and human resources. He is currently a data scientist at Offerzen.
