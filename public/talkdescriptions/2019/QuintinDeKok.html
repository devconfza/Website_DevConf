About the talk

This talk will be about the experiences and lessons learned when Netstar built an on-prem Kubernetes cluster and the lessons learnt when migrating that cluster to Azure Kubernetes Services.

About the speaker

Quintin's been a developer for 20 years in the banking and telematics industries. He's developed multiple high transaction volume systems in C#, .NET and more recently Scala and R. When he's not coding he's spending time with his family and enjoys reading.
